操作チュートリアル

本シミュレータは、実際のドローン操作を再現しています。
MODEボタン / Eキーで「EASY(ANGLE)モード」と「ACRO(RATE)モード」を切り替えられます。
初心者の方は、スティックを離すと水平に戻るEASYモードをおすすめします。

【ホバリング機能】 スティックまたはキー（W, S）から手を離したニュートラル時には、重力と推力が釣り合ってその場で自動的にホバリングします。

【初心者向け：前に直進するコツ】 1. Wキーを少し押して機体を浮かせます。
2. ↑キー(ピッチ) を押し続けて、機体を前に傾けます。
3. 前に傾くと高度が下がるので、Wキーを押し込んで推力をプラスします。
4. 機体の前傾(↑) ＋ 推力(W) の組み合わせで、斜め前にビュン！と安定して直進します。

■ スマホでの操作:
  • ★ STICKボタン で「FLOAT（画面どこでも操作）」と「FIXED（下部固定スティック）」を切り替え可能！
  • 画面左側（左スティック） : スロットル / ヨー (上下で高度調整、左右でヨー旋回)
  • 画面右側（右スティック） : ピッチ / ロール (前後左右に機体を傾ける)
  • ★ 触れたその場が中心になるため、指を見なくても滑らかに操縦可能！
  • ★ 視界確保のため、FLOAT操作中はUIボタンが透明にフェードアウトします。
■ キーボード操作:
  • W / S : スロットル
  • A / D : ヨー (左・右旋回)
  • ↑ / ↓ : ピッチ (前進 / 後退)
  • ← / → : ロール (左移動 / 右移動)
  • 各種ボタン操作：
    E(モード), F(スティック), V(視点), M(マップ),
    L(難易度), K(加速), C(感度), B(ダメージ), T(特訓), R(復帰), G(画質), P(角度)
